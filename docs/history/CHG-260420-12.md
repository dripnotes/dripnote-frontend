# [CHG-260420-12] 원두 상세 페이지 명세 고도화 및 컴포넌트 래퍼 버그 수정 (2026-04-20)

## 1. 목적 (Purpose)

- 원두 상세 페이지(`specs/bean-detail-page.md`)의 명세를 시스템 가이드라인(`spec-guidelines.md`)에 맞춰 검증 기준(DoD) 및 UI 상탯값을 포함하도록 고도화합니다.
- `global.md`의 "레이아웃 래퍼 의무화" 규정에 맞추어 모바일 하단 네비게이션 간섭을 방지할 수 있도록 `page.tsx`에 `PageContainer`를 연동합니다.
- 메인 페이지의 추천 원두 섹션 딥링크가 오류나는 문제를 함께 수정합니다.

## 2. 중간 과정 및 변경 사항 (Process & Deviations)

- **명세 문서 개편**: `BeanDetailHero`, `FlavorProfileSection`, `BrewingGuide`는 물론 이전 명세에서 빠져있던 `BeanInfoTable`에도 공통 양식을 적용하여 목적, 프롭스 구조, 상태, 검증 목록 등을 체계화했습니다.
- **레이아웃 래퍼 적용**: `apps/web/app/(main)/beans/[id]/page.tsx`의 최상단 요소를 `<div className="bg-white min-h-screen">`에서 `<PageContainer>`로 마이그레이션하여 모바일 뷰 여백 일관성을 보장했습니다.
- **링크 포맷 수정**: `apps/web/lib/api/main.ts` 내부의 `mockMainData` 추천 원두 링크 경로에서 `detail/` 세그먼트를 제거(`/beans/${bean.id}`)하여 올바른 상세 페이지로 유입되게 했습니다.

## 3. 결과 (Result)

- `specs/bean-detail-page.md` 정비 완료.
- 원두 상세 페이지가 `PageContainer` 내에서 렌더링되도록 수정 완료.
- 메인 페이지 추천 원두에서 상세 보기 클릭 시 404 없이 정상 라우팅됨.
