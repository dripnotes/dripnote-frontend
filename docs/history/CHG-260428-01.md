# [CHG-260428-01] 원두 상세 페이지 명세 최신화 및 UI 상세 개편

## 1. 작업 개요

원두 상세 페이지의 사용자 경험을 고도화하기 위해 명세서(`specs/bean-detail-page.md`)를 최신화하고, 이를 실제 코드에 정밀하게 반영함. 추출 가이드 제거, UI 텍스트 한국어화, 액션 버튼 레이아웃 및 디자인 개선을 포함함.

## 2. 주요 변경 사항

### 2.1 명세서(`specs/bean-detail-page.md`) 최신화

- **추출 가이드 제거**: 원두 상세 정보 집중도를 높이기 위해 `BrewingGuide` 섹션을 명세에서 완전 삭제.
- **UI 용어 한국어화**: 향미 지표 및 데이터 테이블의 라벨을 한국어(산미, 감미, 바디감, 밸런스, 로스팅 등)로 명세화.
- **구매 버튼 정책 변경**: `purchaseUrl` 존재 여부와 상관없이 항상 노출하며, 모바일에서는 **우측 하단 플로팅 버튼(FAB)** 형태로 배치하도록 명세화.
- **디자인 정책 반영**: 구매하기 버튼의 검은색 배경 적용, 데스크톱에서의 좌측 정렬 배치, 모바일 헤더 버튼의 시인성 강화(`bg-white/90`) 명시.
- **북마크 색상 변경**: 강조색을 `Brand-Amber`에서 `Gray-700`으로 변경.
- **정보 테이블 레이아웃**: `BeanInfoTable`의 정보 항목을 데스크톱/태블릿에서 **2행 3열 그리드**로 표시하도록 변경.

### 2.2 UI 컴포넌트 리팩토링 및 고도화

- **`BeanDetailHero`**:
  - `Button` 컴포넌트(ui-library)를 사용하여 리팩토링 및 모바일/데스크톱 대응.
  - **모바일 UI**: 헤더 북마크/공유 버튼 시인성 개선(`bg-white/90`, `backdrop-blur`), 구매 버튼을 우측 하단 플로팅 버튼으로 이동.
  - **데스크톱 UI**: '구매하기' 버튼을 액션 바 좌측에 배치하고 검은색 배경 적용.
  - 북마크 하이라이트 색상을 `gray-700`으로 변경.
- **`FlavorProfileSection` & `BeanInfoTable`**:
  - 모든 UI 라벨을 한국어로 변경 (산미, 감미, 바디감, 밸런스, 로스팅 / 원산지, 카테고리, 분류 등).
  - `BeanInfoTable`을 전체 너비 구조로 개편하고 정보 항목에 2x3 그리드 적용.
- **`BrewingGuide` 제거**: 명세에서 제외됨에 따라 `page.tsx`에서 컴포넌트 호출을 제거하고 물리적 파일 삭제.

### 2.3 공용 UI 라이브러리(`ui-library`) 개선

- **`Button` 컴포넌트**:
  - `disabled` 상태의 시인성 개선 (`opacity-60`).
  - 비활성화 상태에서 마우스 호버 시 `cursor-not-allowed` 스타일이 나타나도록 수정 (`pointer-events-none` 제거).

## 3. 검증 결과

- [x] 원두 상세 페이지에서 추출 가이드 섹션이 노출되지 않음.
- [x] 모든 향미 지표 및 정보 라벨이 한국어로 정상 표시됨.
- [x] 구매하기 버튼이 모바일에서 플로팅 버튼으로, 데스크톱에서 좌측 정렬된 검은색 버튼으로 정상 노출됨.
- [x] 모바일 헤더의 북마크/공유 버튼이 배경에 상관없이 잘 보임.
- [x] 북마크 클릭 시 중간 회색(`gray-700`)으로 하이라이팅됨.
- [x] 원두 정보 항목이 데스크톱에서 2행 3열 그리드로 깔끔하게 배치됨.
- [x] 비활성화된 공유 버튼에 마우스 호버 시 금지 표시 커서가 나타남.

## 4. 향후 작업 제안

- [ ] **토스트 시스템 도입**: 단순 `alert` 대신 디자인 시스템에 맞는 토스트 알림으로 피드백 고도화.
- [ ] **데이터 보강**: 실제 구매 가능한 원두의 경우 `purchaseUrl`을 데이터베이스/Mock에 보강.
