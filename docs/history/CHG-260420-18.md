# [CHG-260420-18] 원두 상세 페이지 모바일 헤더 및 액션 버튼 배치 조정 (2026-04-20)

## 1. 목적 (Purpose)

- 원두 상세 페이지(`beans/[id]`)와 원두 목록 페이지(`beans`) 간의 디자인 일관성을 확보합니다.
- 모바일 해상도에서 브랜드 아이덴티티를 위해 로고를 명시적으로 노출하고, 주요 액션 버튼의 접근성을 개선합니다.

## 2. 중간 과정 및 변경 사항 (Process & Deviations)

- **모바일 헤더 신규 도입 (`BeanDetailHero`)**:
  - `md:hidden` 클래스를 사용하여 모바일 해상도에서만 보이는 상단 헤더를 섹션 최상단에 추가했습니다.
  - 리스트 페이지와 동일한 스타일의 "Dripnote" 로고(`font-playfair text-2xl`)를 좌측에 배치했습니다.
- **액션 버튼 재배치 (Top-Right Action)**:
  - 기존 이미지 하단에 있던 북마크, 공유, 구매 버튼을 모바일 헤더의 우측 상단으로 이동시켰습니다.
  - 버튼 크기를 `h-10 w-10`으로 소형화하고 `rounded-xl` 스타일을 적용하여 콤팩트한 내비게이션 바 형태를 구현했습니다.
- **상태 동기화**:
  - 상단 헤더의 액션 버튼과 데스크톱 뷰의 액션 버튼이 동일한 `isBookmarked` 상태를 공유하도록 구성했습니다.
- **레이아웃 미세 조정**:
  - 로고 헤더와 Back 버튼 사이의 간격을 조절하고, 모바일 액션 바 영역을 기존 위치에서 제거(`hidden md:flex`)하여 중복 노출을 방지했습니다.

## 3. 결과 (Result)

- 원두 목록 페이지와 상세 페이지 간의 모바일 내비게이션 경험이 통일됨.
- 페이지 진입 시 로고와 핵심 기능을 최상단에서 즉각적으로 인지할 수 있게 됨.

- [x] 모바일 상단 우측의 북마크 버튼 클릭 시 하트 아이콘의 상태 변화(Amber 색상) 확인.
- [x] 모바일에서 기존 하단 액션 바가 사라지고 상단으로 통합되었는지 확인.
- [x] 데스크톱/태블릿 해상도에서는 로고 헤더가 숨겨지고 기존의 넓은 레이아웃이 유지되는지 확인.
- [x] 스펙 문서(`bean-detail-page.md`)에 모바일 헤더 명세 반영 완료.
