# [CHG-260420-14] 원두 상세 페이지 레이아웃 및 타이포그래피 최적화 (2026-04-20)

## 1. 목적 (Purpose)

- 모바일기기에서의 불필요한 상단 여백을 제거하여 첫인상(Above the fold)을 개선합니다.
- 고정된 텍스트 크기 대신 `rem`과 `clamp`를 활용한 유동적 타이포그래피(Fluid Typography)를 도입하여 모든 해상도에서 최적의 가독성을 제공합니다.
- 개별 섹션의 너비 제약을 완화하여 전체 `PageContainer` 프레임에 자연스럽게 녹아들도록 조정합니다.

## 2. 중간 과정 및 변경 사항 (Process & Deviations)

- **상단 여백 조정**: `BeanDetailHero.tsx`에서 모바일 기준 `mt-16`으로 설정되어 있던 외부 여백을 `mt-0`으로 변경하여 헤더와의 간격을 최적화했습니다.
- **유동적 타이포그래피 도입**:
  - `h1`, `h2` 등 주요 타이틀에 `text-[clamp(min, preferred, max)]` 패턴을 적용했습니다.
  - 이를 통해 모바일에서는 컴팩트하게, 데스크톱에서는 웅장하게 텍스트 크기가 화면 너비에 따라 실시간으로 변환됩니다.
- **섹션 너비 확장**: `FlavorProfileSection`, `BeanInfoTable`, `BrewingGuide` 컴포넌트의 `max-w-7xl` 제약을 해제하고 `w-full`로 변경하여 `PageContainer` 내에서 더 시원한 개방감을 주도록 수정했습니다.

## 3. 결과 (Result)

- 모바일 페이지 진입 시 원두 이미지가 상단에 더 밀착되어 정보 밀도 상승.
- 해상도 변화에 따라 텍스트 크기가 부드럽게 스케일링됨.
- 각 콘텐츠 섹션이 화면 전체 너비를 효율적으로 활용함.

## 4. 검증 및 지표 (Verification & Metrics)

- [x] 모바일 뷰포트에서 상단 여백(`mt-0`) 정상 작동 확인.
- [x] 브라우저 리사이징 시 타이틀 텍스트 크기가 유동적으로 변하는지 확인.
- [x] 와이드 모니터 환경에서 `PageContainer` 너비와 섹션 정렬 일치 여부 확인.
