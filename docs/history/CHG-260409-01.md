# 작업 기록 (CHG-260409-01)

## 1. 작업 목적

- 메인 페이지의 '플레이버 노트 탐색' 섹션을 시각적으로 더 매력적이고 직관적인 이미지 카드 UI로 개편하여 서비스의 프리미엄 무드를 강화함.
- "Visual-First" 전략에 따라 텍스트 설명 대신 고감도 이미지를 전면에 배치.

## 2. 상세 작업 내역

### 2.1 데이터 모델 및 Mock 데이터 확장

- `apps/web/lib/api/main.ts`의 `FlavorNote` 인터페이스에 `tasting_image_link` 필드 추가.
- 각 향미(카카오, 복숭아, 자스민, 딸기 등)에 최적화된 고해상도 Unsplash 이미지 URL 매핑.

### 2.2 메인 페이지 명세 고도화

- `specs/main-page.md`를 업데이트하여 이미지 카드 디자인(정사각형, 라운드값, 애니메이션 수치)을 상세히 명시.

### 2.3 컴포넌트 리팩토링 (`FlavorNotes.tsx`)

- **Layout**: 기존 가로 스크롤 방식에서 브라우저 너비에 따른 **반응형 그리드(Responsive Grid)**로 변경.
  - Mobile: 2 cols / Tablet: 3-4 cols / Desktop: 6 cols
- **Stying**:
  - `aspect-square`로 카드 비율 고정.
  - `overflow-hidden` 컨테이너로 이미지 확대 시 프레임 유지.
  - `linear-gradient` 스크림 오버레이를 통한 하단 텍스트 가독성 확보.
- **Animation**: `framer-motion`의 `whileHover={{ scale: 1.1 }}`을 적용하여 부드러운 배경 줌 효과 구현.

## 3. 검증 결과

- **UI/UX**: 정사각형 카드들이 화면 사이즈에 맞게 적절히 배치되며, 호버 시 이미지가 부드럽게 확대됨을 확인.
- **반응형**: 모바일 뷰(2열)에서 데스크톱 뷰(6열)까지 레이아웃 깨짐 없이 유지됨.
- **이미지**: Mock 데이터의 이미지들이 카드 배경으로 정상 로드됨.

## 4. 향후 과제

- 실제 운영 단계에서 Unsplash 외부 링크 대신 정적 에셋(Public Assets) 또는 이미지 서버의 정제된 이미지로 교체 필요.
