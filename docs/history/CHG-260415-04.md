# [CHG-260415-04] 원두 정보 페이지(/beans) 스펙 작성 및 구현 (2026-04-15)

## 1. 목적 (Purpose)

`/beans` 원두 정보 탐색 페이지를 신규 개발합니다. 아로마 대표 식재료 이미지와 추출 배경 색상으로 원두의 개성을 시각적으로 표현하며, 커피 프로파일(향·맛·바디·로스팅) 기반 필터링과 키워드 검색 기능을 제공합니다.

## 2. 중간 과정 및 변경 사항 (Process & Deviations)

### Rule 29 적용

구현 전 `specs/beans-page.md`를 먼저 작성하고, 가격 표시 안 함(`price` 필드 제거) 결정을 스펙에 선반영한 후 구현을 시작했습니다.

### 주요 설계 결정

| 항목                   | 결정 내용                                                                                |
| ---------------------- | ---------------------------------------------------------------------------------------- |
| **가격 표시**          | 제거 — 정보 탐색 서비스와 커머스 서비스를 명확히 구분                                    |
| **아로마 색상 시스템** | `globals.css` CSS 변수 + `@theme` 토큰으로 `bg-aroma-*` 9종 관리                         |
| **필터링 방식**        | 클라이언트 사이드 (`applyBeanFilters` 유틸 함수), 백엔드 연동 시 서버 사이드로 전환 예정 |
| **카드 이미지**        | `object-contain` — 식재료 전체가 보이도록 (레퍼런스 이미지 참조)                         |
| **Mobile 필터**        | Drawer 패턴 (하단 → 상단 슬라이드, `AnimatePresence`)                                    |
