# [CHG-260420-11] 원두 상세 페이지(Detail) 신규 UI 구현 (2026-04-20)

## 1. 목적 (Purpose)

- 사용자가 특정 원두의 향미 정보와 추천 추출 가이드를 상세하게 확인할 수 있도록, 명세서(`specs/bean-detail-page.md`)에 기반한 신규 페이지 라우트 및 컴포넌트를 추가합니다.

## 2. 중간 과정 및 변경 사항 (Process & Deviations)

- **신규 라우트 생성**: `apps/web/app/(main)/beans/[id]/` 경로를 구축하고 `page.tsx`, `not-found.tsx`를 구현하여 데이터가 없을 때 404 분기 처리가 정상 작동하도록 했습니다.
- **컴포넌트 분리**: 페이지 내의 복잡한 UI를 기능 단위(히어로, 향미 지표, 메타 정보, 추출 가이드)의 서브 컴포넌트(`_components/`)로 분리하여 코드 가독성과 유지보수성을 극대화했습니다.
- **디자인 토큰 적용**: `VisualCard`와 동일한 프리미엄 감성(고해상도 이미지, Glassmorphism, 텍스트 스크림)을 적용하고, `RatingScale`을 재사용하여 명세의 "Analytical Elegance" 무드를 구현했습니다.
- **동적 SEO**: Next.js의 `generateMetadata`를 사용하여 원두별 고유 제목과 설명을 생성하도록 조치했습니다.

## 3. 결과 (Result)

- `BeanDetailHero` 추가: 대표 아로마 썸네일과 기본 액션(북마크, 공유) 제공.
- `FlavorProfileSection` 추가: 5가지 핵심 항목(산미, 단맛, 쓴맛, 바디, 로스팅)을 `RatingScale`로 표기.
- `BeanInfoTable` 추가: 원산지, 고도, 품종 등을 미니멀하게 정리.
- `BrewingGuide` 추가: 추천 추출 레시피를 격자/실험실 무드로 배치.
