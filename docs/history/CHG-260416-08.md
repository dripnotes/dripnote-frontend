# [CHG-260416-08] 필터 칩(Chip) 컴포넌트 웹 접근성(aria-pressed) 개선

## 1. 개요 (Overview)

필터 탐색 중 다중 선택이 가능한 아로마 및 로스팅 칩의 토글 상태를 스크린 리더 등 보조 공학 기기가 인식할 수 있도록 `aria-pressed` 속성을 도입했습니다.

## 2. 상세 변경 내역 (Detailed Changes)

### 2.1 Chip 컴포넌트 ARIA 속성 추가

- **대상 파일**: `BeanFilterPanel.tsx`, `BeanFilterDrawer.tsx`
- **변경 사항**: `motion.button` 엘리먼트에 `aria-pressed={active}` 속성을 추가했습니다.
- **기대 효과**: 사용자가 특정 필터를 활성화했는지 여부를 음성으로 명확히 인지할 수 있게 되었습니다.

### 2.2 스타일 정합성 유지

- 작업 중 발생한 스타일 혼선(Panel 스타일이 Drawer 스타일로 덮어씌워진 현상)을 발견하여 즉시 복구하고, 각 컴포넌트 고유의 디자인 시스템(Panel은 테두리 강조, Drawer는 배경색 강조)을 유지하도록 조치했습니다.

## 3. 검증 결과 (Verification)

- [x] 브라우저 인스펙터(DOM)를 통해 필터 클릭 시 `aria-pressed="true"`와 `"false"`가 실시간으로 전환되는 것을 확인했습니다.
- [x] 데스크톱(Panel)과 모바일(Drawer) 환경 모두에서 동일한 접근성 표준이 적용됨을 확인했습니다.
