# [CHG-260420-01] beans.ts 내 필터 데이터 복사 로직 최적화 (2026-04-20)

## 1. 목적 (Purpose)

- `JSON.parse(JSON.stringify())`를 사용한 무거운 깊은 복사 로직을 전개 연산자(`...`)로 대체하여 런타임 오버헤드를 줄입니다.
- 고정된 객체 구조에 맞는 명시적 복사 방식을 통해 가독성 및 타입 안정성을 개선합니다.

## 2. 중간 과정 및 변경 사항 (Process & Deviations)

- **심층 복사 보장**: `DEFAULT_FILTERS`가 중첩된 `flavor` 객체를 포함하고 있으므로, 얕은 복사로 인한 사이드 이펙트를 방지하기 위해 중첩된 객체까지 명시적으로 전개하여 복사하였습니다.
- **영향 범위 확인**: 해당 함수(`decodeParamsToFilters`)를 호출하는 모든 지점에서 데이터 오염 없이 정상적으로 동작함을 확인하였습니다.

## 3. 결과 (Result)

- `apps/web/lib/api/beans.ts` 내의 복사 로직 최적화 완료.
- 불필요한 직렬화/역직렬화 과정 제거로 성능 향상.
