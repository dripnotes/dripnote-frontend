# [CHG-260420-09] VisualCard 라이브러리 인터랙션 고도화 및 버그 수정 (2026-04-20)

## 1. 목적 (Purpose)

- `VisualCard` 컴포넌트의 미구현 인터랙션(`glow`)을 활성화하고, CSS 트랜스폼과 Framer Motion(`whileHover`) 간의 간섭 문제를 해결하여 안정적인 애니메이션 환경을 구축합니다.

## 2. 중간 과정 및 변경 사항 (Process & Deviations)

- **Root 컴포넌트 개선**:
  - `hoverEffect="glow"` 요청에 따라 Tailwind의 커스텀 섀도우를 이용한 글로우 효과를 구현했습니다.
  - `whileHover` 속성을 명시적으로 추출 및 병합하여 `translate` 효과를 Framer Motion이 단일 제어하도록 수정했습니다.
- **VisualImage 컴포넌트 개선**:
  - 기존의 `scaleClass` (Tailwind 클래스 기반 확대) 방식을 제거하고, `whileHover: { scale: hoverScale }` 방식으로 통합했습니다.
  - 사용자의 요청에 따라 `scale-115`와 관련된 수동 클래스 매핑 수정은 제외하였으나, Framer Motion 통합 과정에서 부적절한 클래스 사용 자체가 제거되어 근본적인 문제가 해결되었습니다.
- **타입 안정성 확보**: `VisualCardImageProps`가 Framer Motion의 속성을 올바르게 상속받도록 인터페이스를 정비했습니다.

## 3. 결과 (Result)

- `packages/ui-library/components/VisualCard.tsx` 리팩토링 완료.
- 애니메이션 충돌 방지 및 인터랙션 유연성 확보.

## 4. 검증 및 지표 (Verification & Metrics)

- [x] `hoverEffect="glow"` 적용 시 그림자 효과 정상 동작.
- [x] `translate` 및 `scale` 애니메이션이 중첩되거나 끊기지 않고 부드럽게 작동.
- [x] 타입 스크립트 빌드 오류 없음 확인.
