# [CHG-260415-03] 로그인 페이지 복귀 링크 UX 개선 (2026-04-15)

## 1. 목적 (Purpose)

`LoginHeader`의 메인 복귀 링크("Main")가 가독성이 낮고 렌더링 시점이 너무 늦어 사용성이 떨어지는 문제를 개선합니다.

## 2. 중간 과정 및 변경 사항 (Process & Deviations)

사용자 피드백 기반으로 다음 방향으로 결정:

- 미니멀 스타일 유지 (글라스모피즘 pill 배경 없음)
- ChevronLeft 아이콘 추가로 직관적인 "뒤로 가기" 의미 강화
- 딜레이를 대폭 단축하여 페이지 로드 직후 즉시 노출

## 3. 결과 (Result)

| 항목            | 변경 전          | 변경 후                      |
| --------------- | ---------------- | ---------------------------- |
| 렌더링 딜레이   | `delay: 1.2s`    | `delay: 0.2s`                |
| 애니메이션 길이 | `duration: 0.6s` | `duration: 0.4s`             |
| 아이콘          | 없음             | `ChevronLeft` (lucide-react) |
| 텍스트          | `main`           | `Main`                       |
| 폰트 크기       | `text-xs`        | `text-sm`                    |
| 투명도          | `text-white/40`  | `text-white/65`              |

## 4. 검증 및 지표 (Verification & Metrics)

### 테스트 전략 및 항목

- [x] 로그인 페이지 로드 시 0.2s 후 복귀 링크 노출 확인
- [x] ChevronLeft 아이콘과 "Main" 텍스트가 flex gap-1로 정렬됨
- [x] 투명도 /65로 어두운 배경 대비 가독성 향상 확인
- [x] hover 시 /90으로 자연스러운 강조 확인
