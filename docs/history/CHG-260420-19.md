# 작업 이력 (CHG-260420-19)

## 1. 작업 개요

- **작업명**: 향미 지표 체계 현대화 및 5단계 표준 척도 표준화
- **작업 번호**: `CHG-260420-19`
- **담당 AI**: Antigravity
- **날짜**: 2026-04-20

## 2. 작업 목적

- 원두의 감각적 특성 표현에서 부정적 뉘앙스를 줄이기 위해 '쓴맛(Bitterness)' 지표를 제거하고 '밸런스(Balance)' 지표를 신규 도입합니다.
- 조각나 있던 지표 척도(로스팅/바디 3단계, 그 외 5단계)를 모든 지표에 대해 **5단계(1-5) 표준 척도**로 통일하여 데이터 일관성 및 시각적 위계를 강화합니다.
- 데이터 모델, 목 데이터, 필터 UI, 상세 페이지 UI, 명세서를 일괄 업데이트하여 서비스 전체의 정합성을 확보합니다.

## 3. 상세 작업 내용

### 3.1 데이터 레이어 및 인터페이스 개편 (`lib/api/`)

- `beans.ts`:
  - `BeanInfo` 및 `BeanFilterState` 인터페이스에서 `bitterness`를 `balance`로 교체.
  - `RoastingType` 및 `BodyType`을 1~5 단계로 확장.
  - `mockBeansData` 내 모든 원두의 향미 데이터를 5단계 척도에 맞춰 재분배(Redistribution).
  - 필터링 로직 및 URL 파라미터 인코딩/디코딩 로직을 신규 체계에 맞게 수정.
- `main.ts`:
  - `RecommendedBean` 인터페이스 및 `mockMainData` 매핑 로직에서 `bitterness`를 `balance`로 교체.

### 3.2 UI 컴포넌트 업데이트

- `BeanCard.tsx`: 호버 오버레이 내 지표를 `Balance`로 교체하고, 모든 인디케이터의 `max` 값을 5로 통일.
- `FlavorProfileSection.tsx`: 원두 상세 페이지 내 지표 레이아웃 조정 및 5단계 척도 반영.
- `BeanFilterPanel.tsx` & `BeanFilterDrawer.tsx`:
  - 쓴맛 필터 제거 및 밸런스 필터 추가.
  - 바디감 및 로스팅 필터를 5단계로 확장하고 단계별 설명(매우 가벼움~매우 묵직함 / Light~Dark 5단계) 업데이트.

### 3.3 명세서(Specs) 동기화

- `common-ui-spec.md`: 표준 지표 개편 사실 및 5단계 표준 척도 도입 명시.
- `bean-detail-page.md`: 상세 페이지 내 지표 레이아웃 및 척도 변경 사항 반영.
- `beans-page.md`: 필터 시스템 및 카드 오버레이 명세 최신화.
- `main-page.md`: 추천 섹션 카드 내 지표 체계 반영.
