# 작업 기록 (CHG-260413-01)

## 1. 목적 (Purpose)

- 메인 페이지의 '플레이버 노트 탐색' 섹션 구현 과정에서 발견된 이미지 부정합(Incorrect Mapping) 및 부적절한 경로 이슈를 제보받아 이를 수정함.
- "Visual-First" 전략의 완결성을 위해 각 플레이버 항목에 가장 부합하면서도 프리미엄 무드를 유지할 수 있는 고해상도 이미지를 재선정함.

## 2. 중간 과정 및 변경 사항 (Process & Deviations)

- **이미지 검증**: 기존 '포도(Grapes)' 항목의 이미지가 실제로는 꽃다발(Bouquet) 이미지임을 브라우저 대리인을 통해 확인함.
- **이미지 및 항목 명칭 수정**: 사용자가 제공한 6종의 정확한 Unsplash 원본 링크를 적용하고, '밀크 초콜릿'을 '아몬드'로, '포도'를 '자몽'으로 변경하여 서비스 컨셉을 정교화함. 추가로 '캐러멜'과 '스모키' 2종을 신규 추가함.
- **레이아웃 최적화**: 화면 크기에 따라 데스크톱(4열), 태블릿(3열), 모바일(2열)로 격자 구조를 재배치하여 시각적 안정성을 확보함.
- **경로 최적화**: 쿼리 파라미터가 포함된 전체 URL 형식을 그대로 사용하여 이미지 로딩 신뢰성을 확보함.

## 3. 결과 (Result)

- `apps/web/lib/api/main.ts`: 모든 플레이버 노트의 항목명 및 이미지 링크를 사용자 요청에 맞춰 최종 업데이트 완료.
- `specs/main-page.md`: 이미지 선정 사유 및 항목명 변경 이력을 명세서에 기록하여 정합성 유지.

## 4. 검증 및 지표 (Verification & Metrics)

### 테스트 전략 및 항목

- [x] 각 플레이버 노트 클릭/호버 시 해당 항목에 부합하는 이미지가 노출되는지 시각적으로 검증.
- [x] '밀크 초콜릿' -> '아몬드', '포도' -> '자몽'으로 항목명이 정상 변경되었는지 확인.
- [x] Unsplash 이미지 서버로부터 직접 제공되는 고화질 에셋이 로딩 지연 없이 표시되는지 확인.
