# [CHG-260420-05] 명세서(Specs) 최신화 및 아키텍처 정합성 정비 (2026-04-20)

## 1. 목적 (Purpose)

- 실제 구현 완료된 Phase 8의 변경 사항(RatingScale 추출, BeanCard 공통화 등)을 명세서(`specs/`)에 반영하여 문서와 코드 간의 괴리를 제거합니다.

- **중복 제거**: `main-page.md`에 파편화되어 있던 `BeanCard` 명세를 제거하고 `beans-page.md`를 참조하도록 변경하였습니다.
- **인터페이스 동기화**: `Roasting` 단계(1~3) 및 향미 프로필 Props 등을 최신 구현 코드를 기반으로 모든 명세서에 업데이트하였습니다.

## 3. 결과 (Result)

- `specs/beans-page.md` 업데이트 완료.
