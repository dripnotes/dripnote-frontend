# [CHG-260408-02] 카피라이트 문구 통일 및 푸터 디자인 조정 (2026-04-08)

## 1. 목적 (Purpose)

- 서비스 내 모든 페이지의 카피라이트 문구를 하나로 통일하여 브랜드 일관성을 확보합니다.
- 명세서(`specs/common-ui-spec.md`)의 최신화와 실제 구현물의 정합성을 맞춥니다.

## 2. 중간 과정 및 변경 사항 (Process & Deviations)

- **초기 시도**: 모든 페이지에 'Premium Dark Minimal' 디자인을 적용하기 위해 푸터 배경을 `Primary-Surface`로 변경하고, 대문자와 넓은 자간을 사용하여 현대적인 감각을 시도했습니다.
- **피드백 및 수정**: 변경된 디자인이 기존의 담백한 브랜드 아이덴티티와 다소 이질적이라는 사용자 피드백을 수용하여, 원래의 라이트 테마 스타일(로고 포함)로 복구했습니다.
- **최종 결정**: 디자인은 익숙하고 깔끔한 스타일로 유지하되, 문구는 `© 2026 Dripnote. All rights reserved.`로 완벽히 통일하였습니다. `uppercase` 속성을 제거하여 가독성과 친근함을 높였습니다.

## 3. 결과 (Result)

- `Footer.tsx`: 라이트 테마 스타일 복구 및 통일된 카피라이트 적용.
- `login/page.tsx`: 하단 카피라이트에서 'Lab' 제거 및 메인 페이지와 동일한 텍스트 스타일 적용.
- `specs/common-ui-spec.md`: 사용자의 선호도와 피드백을 반영하여 푸터 디자인 명세 업데이트.

## 4. 검증 및 지표 (Verification & Metrics)

### 테스트 전략 및 항목

- 브라우저 서브에이전트를 사용하여 메인 페이지와 로그인 페이지 하단 푸터의 시각적 일관성을 검증함.
- [x] 모든 페이지 카피라이트 문구 일치 여부 확인
- [x] Uppercase 스타일 제거 확인
- [x] 푸터 라이트 테마 복구 확인
