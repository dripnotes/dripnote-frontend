# [CHG-260420-07] 원두 상세 페이지(Detail) 명세 작성 및 데이터 확장 (2026-04-20)

## 1. 목적 (Purpose)

- 원두 탐색 페이지에서 특정 원두를 선택했을 때 연결되는 상세 페이지(`/beans/[id]`)의 설계를 명문화합니다.
- 상세 페이지 구현에 필요한 추가 데이터 필드(로스터리, 추출 레시피 등)를 정의하고 Mock 데이터를 보완합니다.

## 2. 중간 과정 및 변경 사항 (Process & Deviations)

- **Mock Data 및 인터페이스 확장**: `BeanInfo` 인터페이스에 `roastery`, `processing`, `recipe` 등의 필드를 추가하고 `mockBeansData`에 실제 데이터를 보완하였습니다.
- **신규 명세서 작성**: `specs/bean-detail-page.md`를 신설하여 상세 페이지의 레이아웃, 컴포넌트(`BeanDetailHero`, `FlavorProfileSection`, `BrewingGuide` 등) 및 SEO 요건을 정의하였습니다.
- **기존 명세서 정합성 갱신**: `beans-page.md` 및 `main-page.md` 내의 상세 페이지 링크 경로를 `/beans/{id}`로 통일하고 미결 사항을 업데이트하였습니다.

## 3. 결과 (Result)

- `specs/bean-detail-page.md` 생성 완료.
- `apps/web/lib/api/beans.ts` 데이터 확장 완료.
- 페이지 간 네비게이션 경로 일관성 확보.
