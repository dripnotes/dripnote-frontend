# [CHG-260420-06] VisualCard.Image asChild 사용 시 중복 Props 제거 (2026-04-20)

## 1. 목적 (Purpose)

- `VisualCard.Image` 컴포넌트에서 `asChild` 속성을 사용하여 `next/image`를 렌더링할 때, 부모와 자식에 중복으로 정의된 `src`, `alt` 속성을 정리하여 데이터 흐름의 일관성을 확보하고 불필요한 속성 전달을 방지합니다.

## 2. 중간 과정 및 변경 사항 (Process & Deviations)

- **AromaCard 수정**: `VisualCard.Image`에서 `src`, `alt`를 제거하고 내부의 `next/image` 속성만 유지하도록 변경하였습니다.
- **BeanCard 수정**: 동일한 패턴이 발견된 `BeanCard` 컴포넌트에서도 중복 속성을 제거하였습니다.
- **패턴 표준화**: `asChild` 사용 시 실제 렌더링을 담당하는 자식 컴포넌트가 속성의 Single Source of Truth(SSOT)가 되도록 규정하였습니다.

## 3. 결과 (Result)

- `apps/web/components/common/cards/AromaCard.tsx` 수정 완료.
- `apps/web/components/common/cards/BeanCard.tsx` 수정 완료.
- 코드 정합성 및 가독성 향상.

## 4. 검증 및 지표 (Verification & Metrics)

- [x] `VisualCard.Image`에 `asChild` 적용 시 이미지 렌더링 정상 동작 확인
- [x] 호버 애니메이션(`hoverScale={1.1}`) 정상 동작 확인
