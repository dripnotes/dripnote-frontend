# [CHG-260416-12] 명세서·작업 이력 문서 정비 및 드로어 UI 고도화

## 1. 개요 (Overview)

프로젝트가 고도화됨에 따라 실제 구현과 차이가 발생한 명세서(`specs/beans-page.md`)와 작업 이력(`docs/history/`) 문서들을 정비했습니다. 또한 유저 요청에 따라 모바일 드로어를 전체 화면 높이로 확장하고, 확인이 어려운 시각적 검증 지표(이미지/영상)를 문서에서 제거하여 사실 중심의 텍스트 기록으로 일원화했습니다.

## 2. 상세 변경 내역 (Detailed Changes)

### 2.1 UI/UX 개선

- **BeanFilterDrawer 높이 확장**: 모바일 드로어 높이를 `h-[96vh]`에서 `h-full`로 변경하여 전체 화면을 사용하는 몰입감 있는 레이아웃을 구현했습니다.
- **드로어 인터랙션 정정**: 상단 닫기 버튼 대신 '취소하기' 버튼과 드래그/Backdrop을 통한 닫기 방식을 명세서 및 관련 문서에 반영했습니다.

### 2.2 명세서(specs/beans-page.md) 정합성 확보

- **가격 노출 정책 일원화**: 카드 및 필터 섹션에서 "가격" 언급을 모두 제거하고 '비노출' 정책으로 명세를 통일했습니다.
- **컴포넌트 상태 명시**: `BeanSearchBar`, `BeanFilterPanel` 등이 로컬 상태(`localValue`, `localFilters`)를 관리하며 적용 시점에만 상위로 전달함을 명시했습니다.
- **마크다운 품질 개선**: API 예제 블록에 `http` 언어 식별자를 추가하고 오타를 수정했습니다.

### 2.3 작업 이력 및 관련 문서 클리업

- **절대 경로 수정**: `file:///` 형식의 절대 경로 링크를 레포지토리 상대 경로로 전환하여 범용적인 링크 연결을 지원합니다.
- **구현 사실 정정**: 과거 이력 중 "sticky" 레이아웃을 "flex + shrink-0"으로, "debounce"를 "Enter-triggered"로 실제 구현에 맞게 수정했습니다.
- **시각적 지표 제거**: `walkthrough.md` 및 모든 히스토리 문서에서 `verify_*.webp` 이미지와 "영상을 통해 확인" 등의 문구를 제거했습니다.

## 3. 검증 결과 (Verification)

- [x] 모바일 드로어가 100% 높이로 정상 렌더링됨을 확인했습니다.
- [x] 명세서와 실제 코드의 동작 방식(상태 관리, 버튼 레이아웃)이 일치함을 확인했습니다.
- [x] 모든 작업 이력 문서의 링크가 상대 경로로 정상 작동함을 확인했습니다.
