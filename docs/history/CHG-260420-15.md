# [CHG-260420-15] 원두 상세 히어로 반응형 레이아웃 개선 (2026-04-20)

## 1. 목적 (Purpose)

- 원두 상세 페이지의 `BeanDetailHero` 컴포넌트가 태블릿 해상도에서도 데스크톱과 동일한 가로 배열(2컬럼)을 유지하도록 개선합니다.
- 모바일 환경에서 텍스트 정보를 이미지 상단에 오버레이 형태로 배치하여 `BeanCard`와 시각적 일관성을 확보하고 정보 밀도를 높입니다.

## 2. 중간 과정 및 변경 사항 (Process & Deviations)

- **태블릿 그리드 적용**: 그리드 분기점을 `lg:grid-cols-2`에서 `md:grid-cols-2`로 조정하여 768px 이상의 기기에서 좌우 배치가 활성화되도록 했습니다.
- **모바일 보더 및 패딩 조정**: 이미지와 버튼 사이의 가독성을 위해 모바일에서는 `border-t`를 제거하고 패딩(`pt-4`)을 최적화했습니다.
- **텍스트 가독성 확보**: 모바일 오버레이 텍스트의 가독성을 위해 하단 그라데이션(`bg-linear-to-t`)을 적용하고 텍스트 색상을 화이트로 설정했습니다.
- **구조적 분리**: 동일한 정보 영역을 `md:hidden`(모바일 오버레이)과 `hidden md:block`(태블릿/데스크톱)으로 분리하여 각 환경에 최적화된 마크업을 유지했습니다.

## 3. 결과 (Result)

- 태블릿(768px~1023px)에서 이미지와 원두 정보가 좌우로 배치되어 공간 활용도 상승.
- 모바일에서 원두 이름과 로스터리명이 이미지 하단에 일체감 있게 오버레이됨.
- 뒤로 가기 버튼 및 액션 바(버튼)는 기존 스타일 및 위치를 유지하여 사용성 확보.

## 4. 검증 및 지표 (Verification & Metrics)

- [x] 모바일 뷰: 원두 이미지 - 텍스트 오버레이 - 액션 바(하단) 순서 확인.
- [x] 태블릿 뷰: 이미지(좌) - 텍스트 및 액션 바(우) 배치 확인.
- [x] 오버레이 시 그라데이션 적용으로 인한 텍스트 시인성 확인.
