# [CHG-260415-05] 원두 정보 페이지 UI 사용성 개선 (Rating Bar 및 레이아웃 최적화) (2026-04-15)

## 1. 목적 (Purpose)

원두 정보 페이지(`/beans`)에서의 사용자 경험과 시인성을 높이기 위해 스펙 문서 선행 현행화, 헤더 배경 분기 적용, 검색창 위치 재배치, 필터 인터페이스(Rating Bar 표출 방식 도입)를 개선합니다.

## 2. 중간 과정 및 변경 사항 (Process & Deviations)

### Rule 29 적용

문서 수정 사전 검토제와 구현 전 스펙 선반영 의무에 따라, `specs/beans-page.md`를 우선 최신화하고 사용자 확인 과정을 통해 승인받은 후 실제 구현을 진행했습니다.

### 주요 설계 결정

| 항목                   | 결정 내용                                                                                                                                                                            |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **헤더 Glassmorphism** | `usePathname`을 통해 루트 페이지(`/`)에서만 투명 효과가 적용되게 하여, `/beans`에서는 텍스트 시인성을 위한 솔리드 배경을 렌더링합니다.                                               |
| **검색창 이동**        | 검색창을 최상단 `sticky` 영역에서 하단 우측(원두 카드 리스트 상단)으로 옮기고, 공간 낭비를 줄이기 위해 `md:max-w-xs` 너비를 적용했습니다.                                            |
| **Drawer 여백 확보**   | 모바일 Drawer 필터 스롤 시 클릭 영역과 겹쳐지는 문제를 완화하기 위해 스크롤 컨테이너 패딩(`pb-12 px-6`)을 증량했습니다.                                                              |
| **Rating Bar 필터 UI** | 맛(산미, 단맛, 쓴맛)과 바디 필터를 조작 시, `StepSelector` 숫자형 버튼 대신 N단계 막대형(`RatingBar`)으로 시각화하고 값이 높을수록 `bg-amber-200`~`600`으로 짙어지도록 설계했습니다. |
