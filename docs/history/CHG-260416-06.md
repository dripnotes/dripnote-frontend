# [CHG-260416-06] 원두 탐색 페이지 브랜드 로고 추가 및 카드 디자인 리뉴얼

## 1. 개요 (Overview)

원두 탐색 페이지의 브랜드 정체성을 강화하고, 사용자에게 더 감성적이고 시각적인 탐색 경험을 제공하기 위해 페이지 헤더와 카드 디자인을 전면 리뉴얼했습니다.

## 2. 상세 변경 내역 (Detailed Changes)

### 2.1 브랜드 로고 및 헤더 레이아웃 개편

- **로고 추가**: 페이지 상단에 브랜드 로고(`Dripnote`)를 추가했습니다.
- **스타일링**: `font-playfair`, `text-2xl`(1.5rem), `font-bold`를 적용하여 고전적이고 우아한 무드를 형성했습니다.
- **수직 정렬**: 로고와 원두 결과 수를 `flex flex-col`로 배치하여 계층 구조를 명확히 했습니다.
- **네비게이션**: 로고 클릭 시 홈 화면(`/`)으로 이동하도록 링크를 연동했습니다.

### 2.2 BeanCard 시각적 디자인 리뉴얼

- **풀-사이즈 배경 이미지**: 기존의 단색 배경 구조에서 벗어나 아로마 사진이 카드를 가득 채우는 `object-cover` 방식을 도입했습니다.
- **텍스트 오버레이 시스템**:
  - 하단에 60% 높이의 선형 그라데이션(`.bg-gradient-to-t`)을 추가하여 다크 모드 무드를 조성하고 텍스트 가독성을 확보했습니다.
  - 원두 이름에 `font-playfair` 세리프 폰트를 적용하여 프리미엄 카탈로그 느낌을 주었습니다.
- **호버 애니메이션 고도화**:
  - 카드 부유 효과(`y: -6px`)와 동시에 배경 이미지가 미세하게 확대(`scale-110`)되는 입체적인 애니메이션을 추가했습니다.

### 2.3 Mock 데이터 최적화

- **이미지 교체**: 새로 도입된 `object-cover` 레이아웃에 최적화된, 배경이 포함된 감성적인 아로마 사진들로 Mock 데이터를 업데이트했습니다.

## 3. 검증 결과 (Verification)

- [x] 페이지 상단 "Dripnote" 로고 렌더링 및 홈 이동 링크 정상 동작 확인.
- [x] 원두 카드 내 이미지 전체 채움 및 텍스트 오버레이 가독성 양호.
- [x] 마우스 호버 시 카드 부유 및 이미지 스케일링 동시 동작 확인.
- [x] PC/모바일 환경에서 레이아웃 깨짐 없음.
