# 작업 이력 (CHG-260420-21)

## 1. 작업 개요

- **작업명**: RatingScale 반응성 개선 및 필터 레이아웃 전면 개편(영문화/그룹해제)
- **작업 번호**: `CHG-260420-21`
- **담당 AI**: Antigravity
- **날짜**: 2026-04-20

## 2. 작업 목적

- `RatingScale` 컴포넌트의 트랜지션 속도를 200ms로 단축하여 사용자 입력에 대한 기민한 반응성을 제공합니다.
- 필터 인터페이스에서 지표별 독립성을 강화하기 위해 'Flavor' 그룹을 해제하고, 지표 순서를 사용성 관점에서 재배치합니다.
- 글로벌 서비스 지향 및 전문성 강조를 위해 모든 지표 레이블을 영문으로 통일합니다.
- 지표별 시각적 구분을 위해 신규 `teal` 팔레트를 도입하고 컬러 매핑을 재정립합니다.

## 3. 상세 작업 내용

### 3.1 RatingScale 고도화 (`ui-library`)

- **반응성 개선**: `duration-300`을 `200`으로 수정하여 애니메이션 속도 상향.
- **신규 팔레트 도입 (`teal`)**: 밸런스(Balance) 지표 전용 청록색 스케일 추가.
  - Hex: `#CCFBF1`(200) ~ `#134E4A`(600)

### 3.2 필터 컴포넌트 레이아웃 개편 (`apps/web`)

- **지표 그룹 해제**: `Acidity`, `Sweetness`를 개별 섹션으로 독립시켜 각 지표의 중요도 균등화.
- **순서 재배치**: `Acidity` → `Sweetness` → `Body` → `Balance` → `Roasting` 순으로 조정.
- **라벨 영문화**: `산미`, `단맛`, `밸런스`, `바디감`을 각각 `Acidity`, `Sweetness`, `Balance`, `Body`로 변경.
- **지표별 컬러 매핑 적용**:
  - `Balance`: `teal`
  - `Roasting`: `espresso` (기존 `amber`에서 변경)

### 3.3 기타 UI 컴포넌트 동기화

- `FlavorProfileSection.tsx` & `BeanCard.tsx`: 상세 페이지 및 카드 오버레이 내 지표 컬러를 필터와 동일하게 동기화 (`Balance: teal`, `Roasting: espresso`).
