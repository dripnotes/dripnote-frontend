# [CHG-260420-13] BeanCard 호버 애니메이션 고도화 (2026-04-20)

## 1. 목적 (Purpose)

- 원두 카드(`BeanCard`)의 호버 시 정보 노출 애니메이션을 더욱 부드럽고 프리미엄하게 개선합니다.
- 단순히 페이드인 되던 정보를 "상승하며 등장(Slide-up)"하는 효과로 교체하여 시각적 깊이감을 더합니다.

## 2. 중간 과정 및 변경 사항 (Process & Deviations)

- **상승 효과 추가**: `BeanCard` 내부의 상세 정보 오버레이 콘텐츠에 `translate-y-8` -> `translate-y-0` 애니메이션을 추가하여 아래에서 위로 부드럽게 올라오는 인터랙션을 구현했습니다.
- **애니메이션 동기화**: 기존에 아래로 사라지던 텍스트 영역(`VisualCard.Content`)도 위로 사라지게(`-translate-y-2`) 변경하여, 카드 전체의 상승 기류와 방향성을 통일했습니다.
- **라이브러리 기반 보정**: `VisualCard.tsx`의 기본 트랜지션 지속 시간을 300ms에서 500ms로 늘리고 `ease-out`을 기본값으로 설정하여 라이브러리 차원의 부드러움을 확보했습니다.
- **Cubic Bezier 적용**: 프레임워크(`framer-motion`) 레벨에서 `[0.22, 1, 0.36, 1]` 커스텀 이징을 적용하여 "빠르게 시작하고 우아하게 멈춤" 효과를 극대화했습니다.

## 3. 결과 (Result)

- `BeanCard` 호버 시 정보가 부드럽게 "슬라이드 업" 하며 등장함.
- 카드 이동 및 텍스트 전환의 속도와 방향성이 조화롭게 일치함.

## 4. 검증 및 지표 (Verification & Metrics)

- [x] 호버 시 끊김 없는 60fps 수준의 부드러운 전환 확인.
- [x] 텍스트가 사라지는 방향과 정보가 나타나는 방향이 모두 위쪽(Upward)으로 통일되어 인지적 피로도가 낮아짐.
