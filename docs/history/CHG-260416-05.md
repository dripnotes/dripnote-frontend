## 2. 상세 내역 (Detailed Changes)

### 2.1 드래그 핸들(Drag-to-Close) 기능 도입

- **대상**: `BeanFilterDrawer.tsx`
- **변경 사항**:
  - `framer-motion`의 `drag="y"`와 `useDragControls`를 적용하여 상단 핸들바를 통해 드로어를 조작할 수 있도록 구현.
  - `dragConstraints={{ top: 0, bottom: 0 }}`와 `dragElastic={{ top: 0, bottom: 1 }}` 설정을 통해 하단 방향으로의 저항감 없는(1:1 추종) 드래그 및 드롭 시 자동 복귀(Snap-back) 구현.
  - `onDragEnd` 임계값(150px) 설정으로 자연스러운 드래그 종료 처리.

### 2.2 레이아웃 구조 및 스크롤 격리

- **대상**: `BeanFilterDrawer.tsx`
- **변경 사항**:
  - 드로어 전체 컨테이너를 `flex flex-col` 구조로 변경.
  - (Fixed Top): 핸들바 영역을 드래그 트리거로 분리.
  - (Scrollable): 필터 본문 영역에 `flex-1 overflow-y-auto`를 적용하여 핸들바와 버튼이 고정된 상태에서 컨텐츠만 스크롤되도록 격리.
  - (Fixed Bottom): 적용 버튼을 하단 `shrink-0` 영역으로 분리.

### 2.3 레이아웃 및 디자인 최적화

- **대상**: `BeanFilterDrawer.tsx`, `specs/beans-page.md`
- **변경 사항**:
  - 드로어 최대 높이를 `h-[96vh]`로 확장하여 상단 여백을 최소화하고 몰입감 증대.
  - 드로어 상단 곡률을 `rounded-t-[2.5rem]`으로 상향하여 프리미엄한 인상 부여.
  - `dragMomentum={false}` 적용으로 조작 시 관성 효과를 제거하여 즉각적인 반응성 확보.

## 3. 검증 결과 (Verification)

- **드래그 추종**: 드로어 핸들을 잡고 내릴 때 커서를 오차 없이 1:1로 따라옴을 확인.
- **자동 복귀**: 닫기 임계값에 미달한 상태에서 핸들을 놓을 때 즉시 상단(`y: 0`)으로 복귀함을 확인.
- **스크롤 격리**: 긴 필터 항목 탐색 시 상단 핸들과 하단 적용 버튼이 가려지지 않고 고정됨을 확인.
