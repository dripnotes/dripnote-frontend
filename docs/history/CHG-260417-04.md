# [CHG-260417-04] BeanFilterDrawer 모바일 핸들 터치 이슈 수정 (2026-04-17)

## 1. 목적 (Purpose)

- 모바일 환경에서 필터 드로어의 상단 핸들을 드래그할 때 브라우저의 기본 스크롤 제스처와 충돌하여 드로어가 닫히지 않는 현상을 해결합니다.

## 2. 중간 과정 및 변경 사항 (Process & Deviations)

- **터치 제어 최적화**: 핸들 영역(`dragControls.start`가 호출되는 영역)에 `touch-none` 및 `select-none` 클래스를 추가하여 브라우저의 간섭을 제거했습니다.
- **히트박스 확장**: 사용자가 더 쉽게 핸들을 잡을 수 있도록 상하 패딩(`pt-5 pb-3`)을 미세하게 조정했습니다.

## 3. 결과 (Result)

- 모바일 시뮬레이터 및 브라우저 환경에서 핸들을 통한 드래그-투-클로즈(Drag-to-close) 기능이 부드럽게 동작함을 확인했습니다.

## 4. 검증 및 지표 (Verification & Metrics)

### 테스트 전략 및 항목

- **Interaction Test**: 브라우저 서브에이전트를 통해 `touch-action: none` 스타일 적용 여부 및 드래그 동작 확인.
- [x] 핸들 영역 `touch-none` 적용
- [x] 드래그 시 드로어 닫힘 임계값(150px) 정상 작동 확인
