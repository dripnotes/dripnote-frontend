# 작업 이력 (CHG-260420-20)

## 1. 작업 개요

- **작업명**: RatingScale 다중 팔레트(Amber, Espresso) 도입 및 지표별 시각적 차별화
- **작업 번호**: `CHG-260420-20`
- **담당 AI**: Antigravity
- **날짜**: 2026-04-20

## 2. 작업 목적

- `RatingScale` 컴포넌트가 지표의 성격에 맞는 색상을 사용할 수 있도록 다중 팔레트 시스템을 도입합니다.
- 특히 '밸런스(Balance)' 지표에 에스프레소 브라운 계열을 적용하여 시각적 위계를 강화하고 브랜드 아이덴티티를 투영합니다.
- 인디케이터 모드(`indicator`)에서 팔레트별 글로우 효과 및 그라데이션을 적용하여 심미적 완성도를 높입니다.

## 3. 상세 작업 내용

### 3.1 RatingScale 컴포넌트 리팩토링 (`ui-library`)

- **컬러 팔레트 시스템 도입**: `PALETTES` 상수를 통해 `amber`와 `espresso` 팔레트를 정의하고 Hex 기반 색상 스케일을 관리합니다.
- **시각효과 고도화**:
  - `indicator` 변체에서 활성 단계에 팔레트별 색상에 최적화된 글로우(`box-shadow`) 효과 적용.
  - 활성 바 내부에 수평 그라데이션 레이어를 추가하여 입체감 부여.
  - 비활성 단계의 투명도를 조정하여 배경과의 조화 개선.

### 3.2 지표별 팔레트 적용 (`apps/web`)

- **Balance 지표 특화**:
  - 원두 상세 페이지(`FlavorProfileSection.tsx`), 원두 카드 호버 오버레이(`BeanCard.tsx`), 필터 패널 및 드로어에서 밸런스 지표에 `espresso` 팔레트를 적용했습니다.
- **공통 사양 유지**: 산미, 단맛, 바디감, 로스팅 지표는 기존 `amber` 팔레트를 유지하여 활력 있는 무드를 보존했습니다.

### 3.3 디자인 시스템 명세 업데이트

- `common-ui-spec.md`: 새롭게 정의된 `Espresso-Scale` 토큰과 `RatingScale`의 다중 팔레트 사양을 명문화했습니다.

2. **인터랙션 피드백**: 필터에서 평점 선택 시 마우스 호버 및 클릭 시의 애니메이션 효과를 팔레트별로 미세 조정.
