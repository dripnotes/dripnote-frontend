# [CHG-260503-01] 구글 애널리틱스(GA4) 통합 및 분석 인프라 구축 (2026-05-03)

## 1. 목적 (Purpose)

- 서비스의 사용자 방문 및 행동 데이터를 수집하여 데이터 기반의 의사결정을 지원하기 위해 구글 애널리틱스(GA4)를 통합합니다.
- Next.js 15 환경에서 최적화된 방식으로 분석 스크립트를 로드하고 관리합니다.

## 2. 중간 과정 및 변경 사항 (Process & Deviations)

- **패키지 도입**: Next.js 공식 분석 라이브러리인 `@next/third-parties/google`을 사용하여 `layout.tsx`에서 선언적으로 GA를 관리하도록 구현했습니다.
- **환경 변수 구성**: `apps/web/.env.local`에 `NEXT_PUBLIC_GA_ID`를 설정하여 환경별 유연성을 확보했습니다. 해당 변수는 빌드 타임에 클라이언트 번들에 노출되는 공개용 트래킹 ID임을 명시합니다.
- **계획 업데이트**: `PLAN.md`에 Phase 9 단계를 추가하고 완료 상태로 업데이트했습니다.

## 3. 결과 (Result)
