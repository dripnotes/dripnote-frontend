# [CHG-260420-08] BeanCard 호버 디자인 복원 및 RatingScale 고도화 (2026-04-20)

## 1. 목적 (Purpose)

- `BeanCard` 컴포넌트의 호버(Hover) 시 나타나는 맛 지표 디자인을 이전 커밋(`3c93467`)의 프리미엄 스타일(슬림 지표 + 글로우 효과)로 복원합니다.
- 공용 컴포넌트 `RatingScale`이 다양한 시각적 요구사항(입력용 바 vs 정보 표시용 지표)을 수용하도록 확장합니다.

## 2. 중간 과정 및 변경 사항 (Process & Deviations)

- **RatingScale 고도화**: `variant="indicator"`를 추가하여 `h-1` 높이의 슬림한 디자인과 `boxShadow`를 활용한 발광 효과를 구현하였습니다. 또한 Amber-Scale의 점진적 Hex 색상 매핑을 적용하여 시각적 완성도를 높였습니다.
- **BeanCard 복원**: `ProfileIndicator` 내부에서 `RatingScale`의 신규 변체를 사용하도록 수정하여, 공용 라이브러리 사용 규칙을 준수하면서도 의도한 디자인을 회복하였습니다.

## 3. 결과 (Result)

- `packages/ui-library/components/RatingScale.tsx` 수정 완료.
- `apps/web/components/common/cards/BeanCard.tsx` 디자인 복원 완료.
- 브랜드 아이덴티티에 부합하는 프리미엄 호버 인터랙션 재확보.

## 4. 검증 및 지표 (Verification & Metrics)

- [x] 호버 시 슬림하고 빛나는 맛 지표가 정상적으로 노출됨.
- [x] 기존 `RatingScale` (`sm`, `md`) 변체의 디자인 영향 없음 확인.
- [x] `pointer-events-none` 적용으로 지표 영역에서의 불필요한 클릭 방지.
