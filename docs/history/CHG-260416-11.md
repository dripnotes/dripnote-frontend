# [CHG-260416-11] 필터링 로직 주석 정합성 및 기획 의도 확인

## 1. 개요 (Overview)

`beans.ts` 내의 필터링 로직 주석과 실제 구현 코드 사이의 불일치를 확인하고, 유저의 최신 기획 의도("선택된 값만 표현")에 맞게 주석을 정정했습니다.

## 2. 상세 내역 (Detailed Changes)

### 2.1 주석 정합성 수정

- **파일**: `apps/web/lib/api/beans.ts`
- **수정 전**: `// Flavor 필터 (선택된 값 이상인 bean만 표시)`
- **수정 후**: `// Flavor 필터 (선택된 값과 일치하는 bean만 표시)`
- **사유**: 외부 리포트에서는 '값 이상(>=)'의 로직을 제안했으나, 유저는 현재 '선택된 값만' 보여주는 방식을 명시적으로 요청했습니다. 실제 코드 구현(`!==`)은 유저의 의도와 일치하고 있으므로, 오해의 소지가 있는 주석만 바로잡았습니다.

## 3. 검증 결과 (Verification)

- [x] `applyBeanFilters` 함수 내의 `bitterness`, `sweetness`, `acidity` 필터링이 `!==` (엄격한 불일치 체크)를 사용하여 선택된 값만 반환함을 코드 레벨에서 확인했습니다.
