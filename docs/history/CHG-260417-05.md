# [CHG-260417-05] 원두 탐색 페이지 로고 노출 반응형 제어 (2026-04-17)

## 1. 목적 (Purpose)

- 원두 탐색 페이지 상단에 노출되는 로고 텍스트가 데스크톱의 전역 헤더와 중복되는 문제를 해결합니다.
- 모바일 환경에서만 로고를 노출하여 브랜딩을 유지하고, 데스크톱에서는 정보 밀도를 최적화합니다.

## 2. 중간 과정 및 변경 사항 (Process & Deviations)

- **반응형 클래스 적용**: `apps/web/app/(main)/beans/page.tsx` 내의 로고 `Link` 컴포넌트에 `md:hidden` 클래스를 추가했습니다.

## 3. 결과 (Result)

- 데스크톱(768px 이상) 브라우저에서는 "Dripnote" 로고가 숨겨지고 결과 개수만 표시됩니다.
- 모바일 환경에서는 기존과 동일하게 로고와 결과 개수가 함께 표시됩니다.

## 4. 검증 및 지표 (Verification & Metrics)

### 테스트 전략 및 항목

- **Visual / Responsive Verification**: 브라우저 서브에이전트 또는 시뮬레이터를 통해 디바이스 너비에 따른 로고 노출 여부를 확인했습니다.

- [x] Mobile 환경 로고 유지 확인
